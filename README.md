# TextCard - 长文转长图

把长文一键生成精美长图，方便在 X / 社交媒体发布。

![TextCard Preview](https://via.placeholder.com/800x400?text=TextCard+Preview)

## 功能特性

### 核心功能

- **文本输入** - 支持富文本输入框，粘贴/手动输入长文
- **模板选择** - 5 套预设样式模板（简约/暗黑/商务/文艺/科技）
- **实时预览** - 右侧实时渲染长图效果
- **自动排版** - 标题/正文/段落自动识别与排版
- **一键导出** - 导出 PNG / JPG，支持选择清晰度（1x / 2x / 3x）
- **水印设置** - 可添加/关闭个人水印（支持位置和透明度调节）

### 体验增强

- **自定义样式** - 字体大小、行间距、内边距、圆角可调
- **背景素材库** - 纯色/渐变/纹理背景
- **Markdown 支持** - 支持 MD 语法输入，自动渲染标题/加粗/列表/代码块
- **字数统计** - 实时显示字数

## 快速开始

### 方式一：直接打开

双击 `index.html` 文件即可在浏览器中打开使用。

### 方式二：本地服务器

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx serve .
```

然后访问 `http://localhost:8080`

## 使用指南

### 基础用法

1. 在左侧「内容编辑」面板输入标题和正文
2. 切换到「样式设置」选择喜欢的模板
3. 调整字体大小、行间距等参数
4. 右侧实时预览效果
5. 点击「导出图片」保存

### Markdown 语法支持

勾选「启用 Markdown 解析」后，支持以下语法：

| 语法         | 效果     |
| ------------ | -------- |
| `# 标题`     | 一级标题 |
| `## 标题`    | 二级标题 |
| `**加粗**`   | **加粗** |
| `*斜体*`     | *斜体*   |
| `- 列表项`   | 无序列表 |
| `> 引用`     | 引用块   |
| `` `代码` `` | 行内代码 |

### 模板说明

| 模板 | 风格描述                     |
| ---- | ---------------------------- |
| 简约 | 白色背景，黑色文字，干净利落 |
| 暗黑 | 深色背景，适合夜间阅读       |
| 商务 | 蓝色调，专业正式             |
| 文艺 | 暖色调，温馨文艺             |
| 科技 | 赛博朋克风，青色点缀         |

### 导出建议

- **清晰度**：推荐使用 2x，适合大多数社交媒体
- **格式**：PNG 支持透明背景，JPG 文件更小

## 技术栈

- **HTML + Tailwind CSS** - 单文件应用，无需构建
- **html2canvas** - 用于导出图片
- **marked.js** - Markdown 解析
- **Google Fonts** - Playfair Display + Noto Serif SC
