# 长文转社媒分享图（GitHub Dark 风格）

一个完全重构优化的纯 HTML/CSS/JavaScript 单页静态工具，将长文本排版为高颜值图片，适合在 X/Twitter 等社媒分享。项目 UI 与配色深度对齐 GitHub 深色主题，并提供多套精美模板、实时预览、高清导出等强大功能。

## ✨ 新版本特性

### 🎨 界面优化
- **现代化UI设计**：全新的界面布局，更加直观易用
- **暗色主题支持**：自动适应系统主题偏好
- **响应式设计**：完美适配桌面端、平板和移动设备
- **交互动画**：流畅的过渡效果和微交互

## 🚀 快速开始

1. **获取项目**
   ```bash
   git clone <repository-url>
   # 或直接下载ZIP文件
   ```

2. **启动应用**
   ```bash
   # 方式1: 直接双击打开 index.html
   # 方式2: 使用本地静态服务器
   python -m http.server 8000
   # 或
   npx serve .
   ```

3. **开始使用**
   - 在左侧面板粘贴文本、调整排版
   - 选择喜欢的模板风格
   - 右侧实时查看效果
   - 点击"生成图片并下载"或使用快捷键 Ctrl+S

## ⚡ 3秒上手（最短路径）

如果你只想快速出图，按这 4 步即可：

1. 粘贴内容到左侧文本框。
2. 模板先选 **“默认”** 或 **“VSCode风格”**。
3. 导出倍率选 **2x**（清晰度和体积平衡）。
4. 点击 **“生成图片并下载”**。

> 小建议：中文长文优先用 Inter / 20px 左右，阅读性更稳定。

## 🎮 使用技巧

### 快捷键
- `Ctrl + S` - 生成并下载图片
- `Ctrl + Shift + C` - 复制图片到剪贴板
- `Ctrl + Shift + R` - 重置所有设置
- `Ctrl + Shift + F` - 切换全屏预览

### 模板推荐
- **正式文档**: 使用"默认"模板，Inter字体，20-22px字号
- **代码分享**: 使用"代码风"模板，Fira Code字体，17-18px字号
- **创意内容**: 使用"玻璃拟态"或"霓虹渐变"模板
- **程序教程**: 使用"VSCode风格"模板，Cascadia Code字体，还原真实编程环境

### 导出质量建议
- **社交媒体**: 2-3倍导出倍率，PNG格式
- **印刷用途**: 4倍导出倍率，PNG格式
- **文件体积**: JPEG格式，质量90%
- **现代浏览器**: WebP格式，更小体积

## 🔧 自定义与扩展

### 添加新模板
1. 在 `styles.css` 中添加 `.template-xxx` 样式块
2. 在 `index.html` 的模板选择下拉中添加对应 `<option>`
3. 在 `app.js` 的 `applyTemplate()` 函数中加入类名

## 🚢 部署建议

### GitHub Pages
1. 将仓库推送到 GitHub。
2. 打开仓库设置 `Settings -> Pages`。
3. Source 选择 `Deploy from a branch`，分支选 `main`（或你的默认分支），目录选 `/ (root)`。
4. 保存后等待 1~2 分钟即可获得线上访问地址。

### Vercel / Netlify
- 这是纯静态项目，无需构建命令。
- 发布目录直接设置为仓库根目录即可。
